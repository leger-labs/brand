What's Included

🎨 Design Tokens

Complete color palette, typography scale, spacing, and shadows. Available as CSS variables, Tailwind preset, and JS module.

🔤 Typography

Geist Sans and Mono fonts with Vercel's precise sizing, line-heights, and letter-spacing.

⚙️ Auto-Generation

GitHub Actions automatically rebuilds design system files when tokens change.

🌙 Dark Mode First

Built on Catppuccin Mocha with optional light mode support.

📦 Submodule Ready

Use as a git submodule in your projects with auto-updates.

🎯 Framework Agnostic

CSS variables, Tailwind preset, or raw JSON tokens. Use what you need.

Usage

Installation

Add the brand kit as a git submodule to your project:

git submodule add https://github.com/leger-labs/brand.git brand
git submodule update --init --recursive

Quick Start

Three ways to use the design system:

1. CSS Variables

@import './brand/dist/fonts.css';
@import './brand/dist/tokens.css';

.btn {
  background: hsl(var(--primary));
  color: hsl(var(--primary-foreground));
}

2. Tailwind Preset

// tailwind.config.js
const preset = require('./brand/dist/tailwind.preset');

module.exports = {
  presets: [preset],
  content: ['./src/**/*.{ts,tsx}'],
};

3. JS Tokens

import { colors } from './brand/dist/tokens.js';

const theme = {
  primary: colors.brand.primary.DEFAULT,
  background: colors.dark.background.primary,
};

Auto-Updates

Set up GitHub Actions to automatically pull brand updates:

name: Update Brand
on:
  schedule:
    - cron: '0 0 * * *'  # Daily
  workflow_dispatch:

jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          submodules: true
      
      - name: Update submodule
        run: |
          git submodule update --remote brand
          git add brand
          git commit -m "chore: update brand" || exit 0
          git push
